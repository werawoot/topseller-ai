# Assets — TopSeller AI

วางไฟล์เหล่านี้ในโฟลเดอร์นี้ก่อน build:

| ไฟล์ | ขนาด | รายละเอียด |
|---|---|---|
| `icon.png` | 1024×1024 px | App icon (PNG, ไม่มี alpha สำหรับ iOS) |
| `splash.png` | 1284×2778 px | Splash screen (PNG) |
| `adaptive-icon.png` | 1024×1024 px | Android adaptive icon foreground |

## แนวทาง Design

- **สีหลัก**: `#EA580C` (orange)
- **Background**: `#FFFDF9` (warm white)
- **Icon**: โลโก้ TopSeller AI บนพื้น orange
- **Splash**: โลโก้กลางจอ บนพื้น `#FFFDF9`

## เครื่องมือฟรีที่แนะนำ

- [Canva](https://canva.com) — ออกแบบ icon/splash
- [EAS Asset Generator](https://github.com/expo/expo/tree/main/packages/expo-asset) — resize อัตโนมัติ
- [AppIcon.co](https://appicon.co) — generate ทุกขนาดจากไฟล์เดียว
